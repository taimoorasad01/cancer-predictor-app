# Breast Cancer Prediction App

## Overview
An interactive machine learning web app that predicts whether a breast 
tumor is malignant or benign based on 30 cell nucleus measurements.
Built with Python, scikit-learn, and Streamlit.

## Features
- 3 ML models: Logistic Regression, Random Forest, SVM
- Interactive sliders for all 30 tumor measurements
- Real-time prediction with probability scores
- Gauge chart showing benign probability
- Model performance comparison in sidebar

## Live Demo
[Click here to use the app](https://taimoorasad01-cancer-predictor-app-app-4tkqpr.streamlit.app/)

## Tech Stack
- Python 3.13
- scikit-learn, NumPy, pandas
- Streamlit, Plotly
- Dataset: Wisconsin Breast Cancer (built into scikit-learn)

## Disclaimer
This app is for educational purposes only and should not be used 
for actual medical diagnosis.
